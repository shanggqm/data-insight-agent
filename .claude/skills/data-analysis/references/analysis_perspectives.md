# 数据分析视角参考

根据数据特点和分析目标选择合适的分析方法。

---

## 一、描述性分析

### 集中趋势

- **均值**：整体水平，易受极端值影响
- **中位数**：稳健中心，适合偏态分布
- **众数**：最常见值，适合分类数据

### 离散程度

- **标准差/方差**：波动幅度
- **变异系数**：相对波动，用于不同量纲对比
- **分位数**：分布形态，识别异常

### 分布特征

- **偏度**：对称性（正偏右长尾，负偏左长尾）
- **峰度**：集中程度（尖峰集中，平峰分散）

---

## 二、时序分析

### 趋势识别

- 移动平均：平滑噪声，显现趋势
- 同比/环比：周期对比
- 增长率：变化速度

### 周期规律

- 小时分布：日内模式
- 星期分布：周内模式
- 月度分布：年内模式

### 异常时点

- 突变检测：急剧变化
- 断点分析：趋势转折

---

## 三、对比分析

### 分群对比

按特征分组，对比差异：

- 组间差异是否显著
- 哪个维度区分度最大
- 交叉分组的交互效应

### 基准对比

- 历史基准：与过去对比
- 目标基准：与预期对比
- 行业基准：与外部对比

---

## 四、关联分析

### 相关性

- **正相关**：同增同减
- **负相关**：此消彼长
- **无相关**：独立变化

注意：相关 ≠ 因果

### 归因分析

- 贡献度拆解：各因素贡献占比
- 因子影响：单因素变化的影响

---

## 五、异常分析

### 统计方法

- **3σ 原则**：超出均值 ±3 标准差
- **IQR 法则**：超出 Q1-1.5×IQR 或 Q3+1.5×IQR
- **孤立森林**：多维异常检测

### 业务判断

- 是否符合业务逻辑
- 是否为数据质量问题
- 是否需要单独分析

---

## 六、分析深度递进

| 层次 | 问题     | 方法     |
| ---- | -------- | -------- |
| 描述 | 是什么？ | 统计汇总 |
| 诊断 | 为什么？ | 下钻拆解 |
| 预测 | 会怎样？ | 趋势外推 |
| 决策 | 怎么做？ | 建议方案 |

---

## 七、常见陷阱

- **辛普森悖论**：整体与分组趋势相反
- **幸存者偏差**：只看到成功样本
- **基数效应**：小基数百分比波动大
- **混淆变量**：第三因素导致虚假相关
