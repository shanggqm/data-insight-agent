# Schema 文档

缓存数据库表结构，避免重复查询。

## 命名规范

```
{source}_{name}.md
```

示例：`mysql_baixiaoying.md`

## 文件模板

```markdown
# {数据源名称} Schema

> 更新时间：YYYY-MM-DD
> 时区设定：UTC / 本地 / 时间戳

## 表清单

| 表名 | 描述 | 常用场景 |
| ---- | ---- | -------- |

## 表结构

### {table_name}

| 字段 | 类型 | 说明 | 索引 |
| ---- | ---- | ---- | ---- |
```

## 使用方式

1. **首次分析**：通过 MCP 工具获取 schema，创建文档
2. **后续分析**：直接读取，跳过探索步骤
3. **发现变更**：及时更新文档
